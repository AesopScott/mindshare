# Contributing To MAPS

MAPS grows by strengthening phase definitions and curating resources that help a specific phase.

## Add A Repo Or Skill

Add catalog entries with enough evidence to make the recommendation auditable:

- Phase
- Fit
- Maintenance signal
- License
- Security concerns
- Teaching value

## Add Or Update A Phase

Keep phase docs practical:

- Purpose
- Inputs
- Activities
- Outputs
- Done criteria
- Teaching exercise

## Add Or Update A Skill

Keep `SKILL.md` concise. Put detailed teaching material in `phases/` and reusable output shells in `templates/`.
