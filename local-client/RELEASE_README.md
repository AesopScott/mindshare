# MindShare Central 0.1.0

Build date: 2026-06-26

## Download Files

- `MindShare Central Setup 0.1.0.exe` - Windows installer.
- `MindShare Central 0.1.0.exe` - portable Windows executable.

## Requirements

- Windows 10 or newer.
- Codex CLI installed and authenticated locally for Codex sessions.
- Claude CLI installed and authenticated locally for Claude sessions.
- Local `.env` files are not included in the download. Tool API keys must be configured on the user's machine.

## Current Notes

- This is an unsigned internal build, so Windows may show a SmartScreen warning.
- The app packages curated MindShare Central UI/content from `local-client/app-content`.
- Runtime sessions, credentials, generated files, and tool auth state remain local to the user's machine.
