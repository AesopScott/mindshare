# Repository Catalog

Use this catalog to map useful repositories to MAPS phases.

## Rating Labels

- Recommended: Strong fit for the phase and suitable for practical use.
- Useful reference: Good ideas or examples, but not necessarily something to adopt directly.
- Experimental: Interesting but immature, narrow, or unstable.
- Avoid: Not maintained, poor fit, risky license, or security concerns.

## Evaluation Fields

| Field | Question |
| --- | --- |
| Fit | Does it help this phase directly? |
| Maintenance | Is it actively maintained? |
| Adoption | Is there evidence people use it? |
| License | Can teams use it safely? |
| Security | Are there obvious security risks? |
| Docs | Can learners understand it? |
| Teaching value | Does it make the phase clearer? |
