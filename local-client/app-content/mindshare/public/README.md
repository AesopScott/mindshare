# MindShare Website Source

This `public` folder is the authored local source for the MindShare website.

Workflow:

1. Make website changes here first.
2. Sync to the Mojo deploy mirror with `scripts/sync-public-to-mojo.ps1`.
3. Publish from `C:\Users\scott\Code\mojo\MindShare` after the local source has the same change.

Do not make MindShare website fixes only in the Mojo deploy mirror. If a production or Mojo-side fix happens first, pull that change back into this folder before continuing local work.
