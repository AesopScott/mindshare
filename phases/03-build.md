# Build

## Purpose

Implement the agent system with prompts, state, orchestration, APIs, tests, and local run instructions.

## Inputs

- System design
- Role definitions
- Tool and data requirements
- Evaluation criteria

## Activities

- Implement the minimal working agent loop.
- Add prompts, routing, state, and orchestration.
- Integrate framework conventions already used by the project.
- Add local verification and developer documentation.

## Outputs

- Working agent implementation
- Local run command
- Initial tests
- Implementation notes

## Done Criteria

- The agent runs locally.
- The implementation matches the design.
- A basic happy path is verified.
- Risks deferred to later phases are documented.

## Teaching Exercise

Build the smallest runnable version of the designed agent and record the command that proves it works.
